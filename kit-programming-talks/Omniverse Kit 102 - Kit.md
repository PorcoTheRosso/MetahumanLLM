# 什么是 Kit？

- 一句话介绍：kit.exe 是 Omniverse 的 python.exe
- Omniverse 应用都是基于 Kit 开发的

# 理解插件

- 万物皆插件
- Create、Code、View 等等等等的本质区别

# 命令行下的 Kit

- kit.exe 在哪里？
- `kit --enable omni.kit.window.console`
- `kit --enable omni.kit.window.extensions`
